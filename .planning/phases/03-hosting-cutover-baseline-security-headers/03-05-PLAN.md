### Step 1: Disable GitHub Pages serving (~1 min)

GitHub repo → Settings → Pages → Source → None → Save.

### Step 2: Lower DNS TTL on existing records (~5 min)

In the registrar UI for `bedeveloped.com` (<REGISTRAR_FROM_PREFLIGHT>), set the existing A record(s) (or CNAME) for `baselayers.bedeveloped.com` to TTL = 300 seconds. Save.

### Step 3: Add custom domain in Firebase Console (~5 min + verification)

Firebase Console → Hosting → Add custom domain → enter `baselayers.bedeveloped.com`.

The Console provides:
- A TXT verification record (e.g. `firebase-site-verification=...`). Add this to the registrar — typically as a TXT record on `baselayers.bedeveloped.com` or on the apex `bedeveloped.com`. Follow whatever scope the Console requests.
- One or more A records (and possibly AAAA) to add for `baselayers.bedeveloped.com`. Note these IPs.
